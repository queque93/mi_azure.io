Microsoft Azure

DEFINITION

Microsoft Azure est la plate-forme de cloud computing de Microsoft, fournissant une grande variété de services que vous pouvez utiliser sans acheter et provisionner votre propre matériel.Azure permet le développement rapide de solutions et fournit les ressources nécessaires pour accomplir des tâches qui peuvent ne pas être réalisables dans un environnement local.

QUELQUES SERVICES

Vue d'ensemble

  1. Services de calcul
    • les machines virtuelles Azure - Linux et Windows
    • les services cloud, les services d'application
    • Batch (pour les travaux de calcul parallèles et par lots à grande échelle), RemoteApp, Service Fabric et Azure Container Service,Bases de données.
  2. Services de stockage
    3. stockage Microsoft Azure (comprenant les services Blob, Queue, Table et Azure Files)

MODELES DE DEPLOIEMMENT

  • le modèle de déploiement «classique»
    • chaque ressource Azure (machine virtuelle, base de données SQL, etc.) était gérée individuellement.
  • Azure Resource Manager
    • introduit en 2014 ,il permet aux utilisateurs de créer des groupes de services associés afin que des ressources étroitement couplées puissent être déployées, gérées et surveillées ensemble.

CONFIDENTIALITE

  • possibilite d'acces du gouvernement aux données même si la société hébergée n'est pas américaine et que les données résident hors des États-Unis
  • creation de Microsoft Azure Trust Center, et Microsoft Azure a plusieurs de ses services conformes à plusieurs programmes de conformité, y compris ISO 27001: 2005 et HIPAA

EXPANSION REGIONAL

généralement disponible dans 54 régions du monde. Microsoft a annoncé l'ouverture prochaine de 12 régions supplémentaires (à compter d'octobre 2018). Microsoft est le premier fournisseur de cloud à très grande échelle à s'engager à construire des installations sur le continent africain avec deux régions situées en Afrique du Sud.
